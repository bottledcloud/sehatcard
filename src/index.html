Records are locked

Enter the passphrase for this device to unlock saved patients. Without it, the records stay encrypted and unreadable.

🛡️

Protect records on this device

Set a passphrase. Saved patient records will be encrypted with it (AES-256-GCM) and can’t be read without it. There is no recovery if it’s lost — write it somewhere safe.

Map terms to SNOMED CT

Attach a coded concept to each clinical term so the record is machine-readable for ABDM. Exact matches are suggested automatically; search to change, or leave a term as plain text.

⚠ Starter codes — verify before clinical use. This is a small convenience subset, not an authoritative value set. Confirm every code against the official SNOMED CT browser / NRCeS, and swap in the real NRCeS value sets for production.
